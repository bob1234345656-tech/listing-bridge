Process

Four steps.
One listing, two platforms.

Built for a single woodworking seller who lists handmade items on both MakerBook and Etsy and wants to stop duplicating work.

01
📋
Paste your listing

Copy the text from your MakerBook listing page and paste it into the tool. The AI extracts the title, description, price, dimensions, and materials automatically.

02
✏️
Review & adjust

Check the extracted fields, choose your Etsy tags from preset woodworking keywords, and set your price markup with a simple slider — defaults to 10%.

03
⚡
Generate

The AI corrects spelling and punctuation in the description without changing your words, generates 13 SEO-optimized Etsy tags, and auto-selects the best category.

04
🏪
Post to Etsy

Use the step-by-step copy panel to paste each field into Etsy, or post directly to your shop as a draft via the Etsy API once connected.

Features

Everything a one-person
woodworking shop needs.

No subscriptions, no accounts, no data collection. Just a tool that runs in your browser.

🖥
Runs locally

The entire app is a single HTML file. Open it in any browser — no install, no server, no internet required for the interface itself.

🤖
Local AI support

Connects to Ollama running on your machine for completely free, private AI processing. No API costs, no data leaving your computer. Also supports Anthropic's cloud API as a fallback.

💰
Automatic price markup

Set a percentage markup for Etsy listings relative to your MakerBook price. Adjustable from 0–100% with a live preview. Defaults to 10% to account for Etsy's fee structure.

🏷
SEO tag generation

Generates up to 13 Etsy-optimized tags based on your listing content, with individual copy buttons for each tag so you can paste them one at a time into Etsy.

📝
Description preserved

Your listing description stays yours. The AI only corrects spelling mistakes and adds missing punctuation — it does not rewrite, reorder, or add any new content.

📦
Step-by-step copy panel

A numbered guide matches Etsy's listing form order. Each field has a one-click copy button that turns green when done, so you always know where you're up to.

Privacy & Data

Your data stays
on your machine.

Built for one seller.

Listing Bridge is a personal productivity tool used exclusively by its developer to manage their own Etsy shop. It is not a commercial product, not a SaaS platform, and not available to the public.

No analytics, no tracking, no advertising. Nothing is collected, stored, or transmitted to any third party other than the Etsy API and optionally the Anthropic API for AI processing.

  • No user accounts or registration
  • No cookies or tracking scripts
  • No data stored on any server
  • AI processing runs locally on your machine via Ollama
  • Etsy OAuth tokens stored only in browser memory for the session
  • Anthropic API key (if used) never leaves your browser tab
  • All listing data remains on your device
Etsy API Integration

Direct posting
to your shop.

The Etsy integration uses OAuth 2.0 with PKCE — the same secure login standard used by major apps. Your Etsy credentials are never seen or stored by this tool.

OAuth Flow
Secure, standard login

Clicking "Connect Etsy" opens Etsy's official login page in a popup. The seller logs in directly on Etsy's servers — this tool never sees their password. After approval, Etsy issues a short-lived access token stored only in browser memory for that session.

The tool uses PKCE (Proof Key for Code Exchange) — the most secure OAuth flow for browser-based applications, as recommended by Etsy's own documentation.

API Permissions Requested
Minimum required scopes
Scope Purpose
listings_w Create and update draft listings in the seller's own shop
listings_r Read existing listings to verify shop connection
shops_r Read shop ID and name to confirm the correct shop is connected

No financial data, order data, customer data, or account credentials are accessed. Listings are always created as drafts — the seller reviews and publishes manually.
